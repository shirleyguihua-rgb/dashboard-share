# 烘鞋器 Q2 仪表盘分享版

这个目录已经是可直接部署到 GitHub Pages 的静态站点。

包含文件：

- `index.html`
- `app.js`
- `styles.css`
- `dashboard-data.json`
- `.nojekyll`

## 部署建议

优先推荐部署到 GitHub Pages。

## 页面说明

- 这是只读版分享页
- 同事可以查看、筛选、排序
- 不能同步飞书
- 不能上传 Excel

## 数据更新方式

每次在主仪表盘点击：

1. `从飞书同步`
2. `发布分享版`

然后重新上传这个目录到 GitHub Pages 对应仓库即可。
